## Make fun banner messages

-   [figlet](https://github.com/patorjk/figlet.js) - Text to ASCII Art Generator
-   [figlet-cli](https://www.npmjs.com/package/figlet-cli) - CLI for figlet
-   [figlet-web](http://patorjk.com/software/taag) - Web Interface for Figlet

```
npx figlet-cli "Hello DevIntersection" | lolcatjs
```

## Get Answers from Stack Overflow

-   [how2](https://github.com/santinic/how2) - Stack Overflow from the terminal

```
npx how-2 -l javascript random hex color
```
