## List most recent local git branches

Install the following `git` alias to show your recent branches

```
git config --global alias.recent-branches '!git for-each-ref --sort=-committerdate --count=10 --format="%(refname:short)" refs/heads/'
cat ~/.gitconfig
```

Then you can use the alias like the following...

```
git recent-branches
master
feature-used-to-work-on
defect-oops-forgot-about-this
```

## Auto run npm tasks on git hooks

Locally install `husky` in your project.

```
npm install husky --save-dev
```

Husky supports all Git hooks found in the [git documentation](https://git-scm.com/docs/githooks)

```
{
  "husky": {
    "hooks": {
      "pre-commit": "npm test",
      "pre-push": "npm run lint",
      "...": "..."
    }
  }
}
```

## Make your git diffs look great

[Good-lookin' diffs](https://github.com/so-fancy/diff-so-fancy)

```
npm install -g diff-so-fancy
git config --global core.pager "diff-so-fancy | less --tabs=4 -RFX"
```

## Visually interact with git via Terminal

-   [tig](https://github.com/jonas/tig) - Text-mode interface for git

```
brew install tig
```

-   [lazygit](https://github.com/jesseduffield/lazygit) - A simple terminal UI for git commands

```
brew install jesseduffield/lazygit/lazygit
brew install lazygit
```
