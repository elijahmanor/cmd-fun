## Safely delete files from the terminal

Move files to macOS trash instead of perminantly deleting with `rm`

```
brew install trash
trash safely-delete-file.js
```

## View a graphical activity monitor

-   [vtop](http://parall.ax/vtop) - Wow such top. So stats. More better than regular top

```
brew install vtop
```

-   [htop](http://hisham.hm/htop/) - An interactive process viewer for Unix systems

```
brew install htop
```

---

## Test your internet speed from Terminal

-   [speed-test](https://github.com/sindresorhus/speed-test)
-   Test your internet connection speed and ping using speedtest.net from the CLI

```
npx speed-test
```

## Use a better cat

-   [bat](https://github.com/sharkdp/bat) - A cat(1) clone with wings


```
brew install bat
```

* Syntax highlighting
* Git integration
* Show non-printable characters
* Automatic paging

## Visually check your disk usage

* [ncdu](https://dev.yorhel.nl/ncdu) - CLI disk usage analyzer

```
brew install ncdu
```

## Quickly navigate folders on the Terminal

* [ranger](https://ranger.github.io/) - ranger is a console file manager with VI key bindings.

```
brew install ranger
```

## View Simple Community-Driven man pages

- [tldr](https://tldr.sh/) - The TLDR pages are a community effort to simplify the beloved man pages with practical examples.

```
npx tldr mkdir
```

## Interactively kill processes

- [fkill-cli](https://github.com/sindresorhus/fkill-cli) - Fabulously kill processes. Cross-platform. 

```
npx fkill-cli
npx fkill 1337
npx fkill safari
npx fkill :8080
```
