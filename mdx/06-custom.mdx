## Function in an Alias

The previous alias couldn't accept an argument, this one can because it defines a function.

```
alias toast='f() { notify -t "Toast" -m ${1:-"Command Complete"} -s Glass };f'
```

Example Usage:

```
npm install && toast                # Message that says "Command Complete" (default)
npm test && toast "Tests complete"  # Message that says "Tests Complete"
```

---

## Standalone Function

Instead of embedding a function in a one-liner alias, you can just define a top-level function instead.

```
function npm-link() {
  echo "Starting to link node_modules for ~/github/$1"
  rm -rf ~/github/$1/node_modules
  npm link ~/github/$1
  echo "Finished linking $1"
}
```

Example Usage:

```
npm-link web-awesome-slice
```

## Create a custom Bash Script

```
#!/usr/bin/env bash

echo "Building..."
rm -rf public/$npm_package_version
pug --obj data.json src/index.pug --out public/$npm_package_version/
node-sass src/index.scss | postcss -c .postcssrc.json | cssmin > public/$npm_package_version/index.min.css
mustache data.json src/index.mustache.js | uglifyjs > public/$npm_package_version/index.min.js
echo "Finished building."
```
