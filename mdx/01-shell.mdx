## Oh-My-Zsh Changing Dirs

With zsh `AUTO_CD` makes it so you don't have to use `cd`

```
$ ~/project-1 > ../project-2
$ ~/project-2 >
```

With zsh use `cd` to replace 2nd argument with 1st occurence of 1st argument in current path

```
$ ~/project-1 > cd -1 -2
$ ~/project-2 >
```

## Switch themes in Oh-My-Zsh

You can change your theme by updating `ZSH_THEME` in `~/.zshrc`, then sourcing the zshrc

[Theme Screenshots](https://github.com/robbyrussell/oh-my-zsh/wiki/Themes)

```
ZSH_THEME="bureau"
```

If you like to live dangerously, you can set a random theme for each instance!

```
ZSH_THEME="random"
ZSH_THEME_RANDOM_CANDIDATES=( "robbyrussell" "agnoster" "avit" "bira" "bureau" "gnzh" "nanotech" "sorin" )
```
